大阪自由行第一天｜心齋橋爆買吉伊卡哇寶寶 & OS Drug超好買 & 梅田藍天大廈夜景

簡介

旅遊時間：2025/10/01 (禮拜三) ~ 2025/10/06 (禮拜一)
此篇記錄 2025/09/30 (禮拜二) ~ 2025/10/01 (禮拜三) 的旅遊過程

大阪自由行正式開跑啦!
這次是我人生第一次挑戰出國，從下班衝去機場飯店開始，一路到大阪的購物天堂心齋橋、平價藥妝聖地OS Drug，最後再用梅田夜景劃下完美句點。
這邊要來分享大阪自由行第一天旅遊行程、購物戰利品、交通與小技巧。

出發前一晚：下班直奔機場飯店

飛機起飛： 早上 06:30
登機時間： 早上 06:00
櫃檯報到時間預估： 早上 04:00

這次旅行訂了紅眼班機 (指在深夜至凌晨起飛) ，因為櫃台托運行李 + 過海關 + 走到登機口以桃園機場來說，保守抓 2 - 3 小時會比較安全，所以我們需要先考慮如何在凌晨抵達機場

方案 說明 優點 缺點 結論
① 睡在自己家，凌晨搭計程車 從新北出發，約 30–60 分鐘車程 省住宿費，彈性高 風險較大：司機遲到或臨時狀況可能誤機 放棄
② 桃園機場膠囊旅館 町草休行館 機場內部設施，離登機口最近 超級方便、不用擔心交通 雙人房滿房、位置有限 原本想選但沒房
③ 桃園凱悅飯店 Hyatt Regency 機場附近高級飯店，搭捷運可直達 舒適、可提前入住、交通便利 價格較高（4200–5500） 最終選擇

一開始想選第一個選項，因為本身住在新北，過去時間約 30 - 60 分鐘，是可以接受的
只是擔心比如預約好計程車，結果司機遲到 (過去遇到過)，或路上發生車禍 (機率小但不排除任何可能)

所以先刪除第一個選項

第二個選項是本來討論決定的方案，只是兩人房沒位置了，不然他在桃園機場是非常的方便

最後選了第三個方案，預約了桃園機場附近的凱悅飯店(Hyatt Regency)，
我們當初提前約 3 個月預定，費用一晚落在 4200 左右，如果出發前一周預定，價格飆升到 5500

這是因為十月開始是一堆連假 (中秋節 , 雙十國慶)，所以大家如果選連假出去一定要提前幾個月預約飯店，不然價格好貴

日期 09/30 (禮拜二) 晚上下班後的交通動線如下

行程段落 交通方式 備註
南港 → 桃園 高鐵 下班後搭乘
桃園高鐵 → 機場旅館站 (A14a) 機場捷運 直達飯店所在站
捷運出口動線 出站左轉 → 飯店櫃檯報到
出站右轉 → 二館（僅供洗衣，無櫃檯）		 請記得出站左轉！ 很多人會走錯

當天看到很多人一出站都右轉去二館，然後發現那邊沒有櫃檯不知道該怎辦，所以大家要記得捷運一出來要左轉

至於為何大家都會走錯，因為左轉過去飯店要走一小段路，大概 1 分鐘左右，而二館就是在捷運站出口隔壁，走過去不用 10 秒，所以很多人會誤會

具體可以參考下圖

  • 紅色：捷運出口
  • 黃色：右轉的二館，這裡是洗衣服的地方，不是飯店報到處，不要去這裡
  • 藍色箭頭：往飯店入住櫃台的地方，這是正確的，一出來要先左轉往這裡走

上圖取自 google map

一進入飯店大廳裝潢看起來蠻氣派，就看到蠻多外國人的

要記得如果要接駁車，辦理入住時，需要先跟飯店人員說明和安排

入住飯店後，到了房間內部裝潢有些老舊

比較特別的經驗是浴室地板有一大灘水，以為是有別人進來或是沒有打掃，就打電話詢問櫃檯，

回覆是近期有漏水的情況，但太晚了我們要趕著睡覺補充體力就繼續住了

洗衣服的地方比較特別是在二館，是投幣式的洗衣機，也就是前面圖片圈起來的黃色區域，要走出飯店過去

洗衣機在二館的地下一樓，裡面有冷氣，放了衣服後可以回到一樓有很多沙發可以坐著等衣服洗好

當初會選擇這間的原因，有一部分是看到他凌晨四點有提供免費的接駁車可以送我們到機場航廈，

只要在指定時間，會有小巴士開到門口接送，司機會先詢問要去哪個航廈並幫忙放置行李到後車廂，

下車時也會協助幫忙把行李拿下來，這部分感受蠻好的。

接駁車 -> 機場花費時間： 5 ~ 10 分鐘

凌晨出發：飛往大阪關西機場

接駁車抵達機場時間：04:12 a.m
搭乘航空：長榮航空

出發流程總覽

流程階段 主要內容 備註／提醒
1️. 抵達機場 依照指示牌前往 Departure（出境） 區 建議預留足夠時間（至少 2–3 小時）
2. 報到櫃檯 長榮航空櫃檯辦理報到或使用 預先報到功能 使用 APP 可直接取得登機證，節省時間
3. 行李托運 出示 護照 + 登機證 建議先看一下自己當前行李箱有多重
4. 行李掃描確認 托運完畢後到後方機器掃描行李 確認畫面上行李無異常再離開
5. 安檢關卡 掃描隨身行李與人身檢查 液體 ≤100ml、行動電源不可托運
6. 護照 + 人臉辨識 掃描護照 → 拍照 → 指紋辨識 若眼鏡反光或髮型遮臉易失敗
7. 登機前等待 通過安檢後即可逛免稅店，等候登機廣播 按 ZONE 區號 分批登機
8. 登機方式 搭乘接駁巴士至飛機，再爬樓梯登機 第一次遇到這樣的方式
9. 起飛階段 系上安全帶、手機開飛航模式 空服員會逐一檢查確保安全
10. 抵達日本 使用 Visit Japan Web 掃碼通關 掃護照 → 指紋 → 人臉拍照（快速方便）

機場報到 & 出境

起飛前 48 小時可以先線上報到，這樣當天會快非常多，我們抵達機場後，因為要拖運行李，要先去櫃檯報到

長榮這邊分了 2 個櫃台

  1. 有先線上報到
  2. 沒有線上登機報到

只要到APP裡點擊報到就可以直接拿到登機證，然後記得要去有完成線上報到的櫃台來排隊等待

當天有告示牌，一開始排錯有點緊張，幸好有問旁邊的服務人員才轉去正確的櫃檯辦理托運行李

托運櫃檯會需要護照+登記證，行李會上去像跑步機的地方秤重，櫃檯上會顯示目前行李箱的重量

我的行李箱帶了26吋的，想說衣服到日本再買就沒放什麼東西，沒想到還是秤了7KG，當下有點驚訝

經濟艙的行李上限是23KG，這樣只剩16KG了呀QQ

行李交出去後，要到最後面有個掃描行李的儀器，確定好自己的行李沒問題後，我們才離開往下一關前進。

往下走，會遇到有個人檢查你的登機證，再往下就是檢查隨身行李的關卡。

隨身行李的限制印象比較深刻的是：

  • 不要帶超過100ML的液體 (要裝在透明夾鏈袋方便檢查)
  • 行動電源一定要放隨身行李不可托運

隨身行李：放到一個塑膠籃子滑過掃描
人的部分：脫外套+口袋不能有東西

接著走過一個檢查儀器，當時我忘記拔掉apple watch想說要不要再重走一次，
但還好當時的人員很好有讓我過，正常來說應該是要拔掉apple watch的。

走過門之後拿好掃描完的隨身行李，就到下一關有個機器要掃描護照，接著拍人臉照片，再按壓雙手食指的指紋。
那因為不知道那時候是沒睡飽還是太緊張，加上沒戴眼鏡高度近視完全看不見，人臉拍照一直失敗，

嘗試拿掉眼鏡+掀劉海都失敗，後來換到最右邊沒人的機器才成功，當時還說萬一再失敗就要走人工審核了，還好通過了！

那通過這邊就可以安心了！接下來的地方就是免稅店可以任意逛，只要到指定的Gate等待登機時間廣播就可以上飛機了。

到登機時間時，並不會馬上就可以登機，航空公司會先廣播讓貴賓先上，再來會按照你登機證的ZONE號碼陸續排隊，
會有長榮的人員舉著一個牌子等到他翻到就可以去排隊登機了

這次經驗也很特別，我們是先上一個小巴士載到飛機附近，
再爬樓梯登機，看來是先幫忙鍛鍊體力為日本做準備呀…

上飛機後的故事

上飛機後，出示登機證座位，門口的空姐會指示你要怎麼走，因為有很多人會放行李到上方，所以會卡著等一下再前進，

到座位後就是繫上安全帶，隨身行李要放到下方移到前面的人座位下，並把手機、手錶調整成飛航模式

為了起飛安全，這期間空姐也會一直幫忙注意，如果沒做到會教你怎麼做。

飛機起飛時沒想像中可怕，會先往前滑行一段時間才開始起飛

起飛的感覺很像搭很快的電梯到高樓層，會有一種耳朵塞住的感覺，打哈欠的話會讓耳朵好一點(?)

餐點也很快就先上了，印象中是肉醬義大利麵、水果、奶油餐包等，送完餐後過陣子才會有空姐推飲料車來問要喝什麼飲料，
我選了可爾必思，後面才知道可以要冰塊。

座位上每個人都會有全新的有線耳機，可以用來配合座位前方螢幕的音樂、電視和電影等等，
飛行時間大約2個小時多，降落時我也覺得沒有很可怕，只有那種騎車騎很快但又不會煞不住車的感覺。

下飛機入境

下飛機後，因為我們有先填寫Visit Japan Web所以通關很快，準備好QR Code到引導的機台，

先掃護照、按指紋，最後才是人臉拍照，體感上我覺得日本的機器比較快速也比較準確！

接著螢幕會顯示一個英文字母拿到d，不知道是什麼意思，再來會按照a-d排隊，

櫃檯上會有個螢幕顯示，按照螢幕排隊就對了。

然後會需要出示護照，這邊建議口罩先拿掉，因為人員會需要幫忙拍照，之後就可以出去等行李

會有個大螢幕寫說你的行李是在哪裡等候，到指定地點等待就好了，這邊拿到行李的時間還蠻快的。
拿到行李之後出去還會有一關要拍照，走過去就結束了，接著就是衝南海電鐵到難波了。

補充: Visit Japan Web

google: Visit Japan Web 就會看到下圖網站，進去填寫資料會得到一個 QR Code ，這個在入境日本的時候會需要

沒有這 qr code 的話，需要在飛機上跟空姐要一張入境表，並用筆填寫，或也可以到日本當地再跟當地的服務人員要紙來寫

用 qr code 比較方便

這邊要特別注意，上圖我用黃色劃起來的網址，最近看到有一些釣魚網站，然後有人被騙信用卡資料

一定要確定網址後面是: `digital.go.jp

像是底下圖這個就是釣魚網站，要小心

基本上看到贊助商廣告 這個出現在 Visit Japan Web 都是釣魚詐騙網站

入住心齋橋：行李寄放

抵達日本關西機場時間：10:15 a.m.
出境時間：10:43 a.m.
搭到南海電鐵時間： 11:05 a.m.

購買南海電鐵

路線：關西機場 -> 難波站

這是到大阪最快的車，這個車需要先買票，我們是用Klook買，選完位置再出發時間前五分鐘可以有兩次機會改票。

我購買的 klook 網址

我回程也是要坐南海電鐵

下圖是我在 2025/10/09 看到的價格

使用方式

購買後要用手機顯示 qr code 進去車站， QR Code 當天才會顯示

然後他沒有手機 app ，也就是你用 klook 購買後會得到一個網址，當天再開這網址，上面顯示的 qr code 去掃描進站

關西機場 -> 搭乘南海電鐵路線

機場出境拿完行李後，坐手扶梯去 2 樓

會看到下圖圈起來的鐵道，跟著箭頭走就對了

跟著上圖指示，走到底會看到下圖圈起來寫著 NANKAI ，這邊開啟手機 QR CODE 過去掃描進站就是南海電鐵

抵達難波站

一出站會看到梅田混合果汁 ，網路上蠻多人在推薦，看到沒有人排隊就跑去買

我覺得蠻好喝的只是有點小杯，但台灣好像比較少喝到這種果汁，不然就是很貴，喝完之後覺得很值得

如下圖，他位置在南海電鐵出口的左手邊，如果你是搭乘南海電鐵，一出站一定會看到

出站後，我們的目的是到心齋橋站

所以打算坐御堂筋線

我們有事前先儲值好西瓜卡，iphone很方便可以直接買西瓜卡，在台灣的時候就可以儲值了。
iphone放到IC卡位置不用解鎖就可以直接刷

難波站去御堂筋線

當初在網路上做功課時，很多文章都表示難波站是大站，走裡面會迷路建議直接到外面馬路行走

但如果你是坐南海電鐵的話，一出站的位置在三樓，可以直接搭乘手扶梯到一樓，然後看一下告示牌寫御堂筋線

跟著一直走其實不會迷路，主要是走外面很熱，裡面有冷氣，所以我第一次抵達難波站，走裡面是沒迷路

告示牌寫得很清楚好懂

飯店放行李

入住飯店：大阪心齋橋靜鐵PREZIO酒店
Check in 時間：15:00 之後
我們當天抵達放行李的時間：13:00

這間飯店位在心齋橋站和本町站的中間，我們從心齋橋站走路過去，因為抵達是下午 13:00 ，所以是先暫時寄放行李

櫃檯人員看到人會主動走出來協助，我覺得很親切，這邊需要房客的護照和訂房人姓名是誰

最後給一個號碼牌，提示下午3點Check-In，到時候帶著護照和號碼牌來領取行李和入住。

經過7-11就買了疲勞減輕的檸檬飲充電，平常不吃酸的我覺得很酸，提神的部分應該是酸醒的(?)

心齋橋PARCO行程：吉伊卡哇寶寶我來啦！

PARCO是一間很大的百貨，也有連通道可以直接到大丸百貨蠻特別的。

吉伊卡哇

6 樓有吉伊卡哇，想要看吉伊卡哇有沒有販售寶寶系列，逛了一圈發現沒看到，後來才發現原來隱藏在角落一區，要抽整理卷時間到了再排隊入場。

這是官方網站 ，有 chiikawa baby 活動

到店外面有個店員會發整理卷，我們中午 13 點多到，抽到的整理卷時間是下午4點15分～4點30分，只有短短的15分鐘可以逛。

消費滿 6600 可以抽底下的杯子，我們抽到兩個都是黃色的烏薩奇

大小很小，真的就是給幼稚園幼童使用的感覺，但很可愛

排隊入場時，他會根據你整理卷的號碼安排你站在地板標示位子，有點像台北寶可夢中心剛開始那樣子。

店員會說商品限購限制，我們剛好當時只有盲盒是每人限制一個，娃娃類限制兩個

因為時間很短，過程中店員也會一邊補貨，朋友說想要甜甜的臉，努力挑了很久最後還好挑到寶寶們都很可愛！

最後還會送栗子寶寶的束口袋!

底下圖片是我們購買的戰利品

HARBS千層蛋糕

聽說他們家的水果千層好吃，剛好很幸運沒有人排隊，就進去內用，我們一開始看菜單不知道哪個是水果千層，直接給店員看圖片，但似乎店員點錯了，如果也想吃水果千層的人要點ミルクレープ這個品項。

我們點了一個水果千層＋栗子千層，水果千層比較好吃，但我個人覺得比較喜歡LADY M的奶油和層次感。

因為低消會限制一人一杯飲料，於是我們點了一杯紅茶和鮮奶茶，我覺得不好喝，如果真的想吃的話感覺外帶回飯店吃比較好!

WPC

這期間逛了一家傘店叫WPC，有賣號稱最強的日傘，剛好有看到一個小八的傘，不會很高調我覺得很好看
!
這間店可以直接開傘看樣子，看完之後放到中間的小籃子，店員會幫忙收回去，只要拿傘交給店員說要買這支的話店員會直接拿全新的給你

有免稅的服務很方便。

OS Drug超好逛！藥妝採購時間

剛好看到附近有OS Drug，這是一家很便宜的藥品店，它不能免稅只收現金，但比免稅還要便宜，這應該是最棒的買藥地方。

買了很多藥品，酸痛凝膠、貼布都幫助了我們這趟日本行程很多，每天走2萬多步晚上擦酸痛藥膏，隔天就完全恢復。

這間店似乎有很多東西是放在櫃檯裡不是擺在外面，有看到中國遊客一直纏著店員問，他們會從櫃檯拿出來。

Check-In 靜鐵飯店

接著時間也差不多回靜鐵飯店check-in，也是遇到同一個櫃檯人員

交給他護照再次告知訂房人名字是誰和行李吊牌，他會幫忙在自助check-in機器操作

接著拿到兩張房卡和一張紙上面會寫房間資訊，往電梯方向有自助區，可以拿盥洗用品，
也有附上塑膠袋子讓你裝，最棒的是有浴鹽，味道會隨天數更換，一開始是果香後面是草木香。

我訂的是高級雙人房，這個房型會有附浴缸，免治馬桶和洗澡的地方是分開的，廁所還很貼心的有附上香氛除臭噴霧，很喜歡日本廁所的馬桶幾乎都有自動沖水，快速放好買完的東西後我們就準備出發到梅田啦！

梅田藍天大廈賞夜景

因為飯店剛好在心齋橋和本町之間，於是我們換成從本町搭往梅田
一樣是御筋堂線往箕面萱野的方向

到梅田站下車後，本來想說一樣按到指示牌想要直接從地下直接走到藍天梅天大廈附近的出口直接迷路…150 公尺的距離走了快一小時。

結果迷路中看到TULLY’S COFFEE，就用GPT翻譯嘗試點了皇家奶茶少冰+蜂蜜好喝！茶很香且濃郁，有點類似台灣SOMA精萃茶歐蕾的感覺

也有經過無印良品，想碰運氣找找火紅的睡眠噴霧，完全失敗好難買呀。

接著我們就努力走出去，從二樓的天橋穿過一個公園，再走過一個很像施工會圍出人行道的地方就到了藍天梅田大廈，剛好也天黑了很適合看夜景

要上去需要從三樓轉電梯直達上去，我們的票有含兩個愛心鎖，所以先出發到白色櫃檯，櫃檯人員很親切會讓我們先選好顏色後，再給我們兩張紙填寫想要刻的字

可以到邊邊有專門的桌子和筆填寫，我們寫了紀念日時間和彼此的英文名字，最後櫃檯人員會再次確認英文大小寫等資訊後會放入一個機器要準備開始刻，
那很好的是會提前和我們說要不要錄影過程，她才按下開始鍵，最後拿出來也會放在一個盒子讓我們拍照，櫃檯人員很可愛，最後還用了中文和我們說謝謝。

最後也有提醒我們，這個鎖如果鎖上了就再也打不開了，所以要謹慎一點，我們決定當紀念品於是帶回去，沒有打算掛在那邊，因為時間久了可能也是被清掉。

接著我們會搭一個手扶梯上去頂樓，這個過程很漂亮，我覺得很值得錄影

當時比較可惜的是前面有人，上去後就會到星光走廊，比較特別的是光照到某些衣服材質都會特別亮

上去的人很多有各國遊客也有日本人，但不會到無法走路的擁擠程度，我覺得很特別的是往建築中心看也很漂亮，夜景也很浪漫可以看整個梅田的樣子。

到這邊的時候我的腰跟腳底已經撐不住，所以這個行程結束後我們就搭車回去飯店了。

出乎意料好吃的飯店旁餃子

回到飯店差不多也很晚，好像剩的餐廳的都是偏居酒屋，我們也累了就選了飯店旁的餃子意外的好吃！

也點了很多其他的菜，自製的麵湯頭也蠻特別的，那時候比較神秘就是後來有一位日本男生坐到我旁邊，

他好像在直播還是什麼一直講話，加上鏡頭有些往我這邊拍，我們交換位子那個男生就安靜的趴著

後面好像不知道是醉了還是睡著了，這個經驗有點微妙，不過整體來說食物是很好吃的，也會有看到很多日本人聚餐應酬。

居酒屋醉漢記得離遠一點
這一條很多居酒屋，晚上有很多怪怪的抽菸人士

結語：第一天的總結

第一天的行程雖然滿檔，但充滿驚喜與滿足感，只是回家算花的錢可能會哭，哈哈哈。
從凌晨出發開始第一次出國過海關、心齋橋快樂大爆買、OS Drug的超值購物行程和梅田藍天大廈的浪漫夜景，整天幾乎都在移動與探索之中發現計劃外的驚喜。
雖然腳快報廢，腰也在罷工，但看到戰利品的那一刻一切都值得了，最後回到飯店寶寶們買太多啦，看到照片突然覺得很像什麼人口販子XD。

整體來說，大阪真的很適合自由行新手，不論是交通、親切的店員，還有購物體驗都很棒。
接下來的幾天也讓人更期待，看看大阪還會帶來什麼驚喜吧！接下來是京都+大阪電器之旅。
