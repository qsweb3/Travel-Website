[2025] 美國舊金山（SFO）轉機全攻略

發表於2025-10-15|更新於2025-10-20|us-guide
|總字數:405|閱讀時間:1分鐘|瀏覽量:

網路大家寫說轉機至少預留 3 小時以上

簡介

飛機：長榮航空 -> 美國聯合航空
路線：TPE -> SFO -> BOS

源於 2025/10 月底要去美國公差一個禮拜

需要在舊金山機場 (SFO) 轉機至波士頓，所以這裡紀錄一下內容

備註： TPE -> SFO 這一段是長榮航空，SFO -> BOS 這一段是美國聯合航空

下飛機 -> 過海關

非美國籍會被移民官問你來的理由，所以這邊要先想一下

然後聽說如果你是做資安的，不可以說跟 Security 相關

會被當成是駭客，然後抓去小房間

上述題外話，反正下飛機之後，因為非美國人，所以要先照著底下告示牌走:Visitors

網路蒐集常見的移民官問題

  • 來美國要幹嘛?
  • 目的地是哪裡?
  • 第一次來美國嗎?
  • 自己一個人嗎? 還有跟誰嗎 ?
  • 你有帶現金嗎?
  • 你有帶食物嗎?

拿行李

去美國只要是從國際線轉國內線，不管是直掛還是同一家航空公司，都需要先領行李再託運

之後要轉機要跟著 Connections 方向走

走一陣子會有人引導再次托運行李

過安檢

這邊就跟台灣差不多，流程就是

  1. 給護照
  2. 照相
  3. 隨身行李檢查

前往登機口

到這邊跟台灣一樣就是看告示牌找登機口，走過去等待即可

備註

美國機場很愛亂換登機口，所以需要隨時注意一下
