【2025最新】Visit Japan Web填寫教學｜日本入境線上申報完整攻略

發表於2025-10-15|更新於2025-12-29|jp-guide
|總字數:1.8k|閱讀時間:5分鐘|瀏覽量:

近年入境日本變得很方便，現在不需要再填寫空姐發的紙本小卡，只要透過日本官方的Visit Japan Web，就能在出發前線上完成所有入境資料登錄。
抵達後，只要掃描QR Code，就能快速通關。

這篇懶人教學將一步步教你從註冊帳號、填寫入境資訊到產生QR Code，全程只要五分鐘，手機操作也沒問題哦！

一、為什麼現在入境日本要用 Visit Japan Web？

Visit Japan Web是日本政府推出的官方線上入境系統，讓外國旅客可以提前完成入境審查、海關申報與免稅登錄等流程。
以往登機時空服員會發入境卡與海關申報單，現在這些都能一次在線上搞定。

使用Visit Japan Web的三大好處：

  1. 節省時間：免填紙本卡片。
  2. 線上完成：手機也可操作，不需印出文件(但是如果擔心手機出意外，還是可以先印出QR Code)
  3. 快速通關：抵達現場掃描QR Code即可入境

建議大家在出發前就先填好，抵達後會省很多時間。

二、準備開始之前，你需要準備的資料：

請先準備以下資訊：

  • 有效護照
  • 航班資訊（航班號碼和抵達日期）
  • 在日本的住宿地址
  • 有效電子信箱

提醒：

  • 原則上一個Email一個帳號
  • 只有同行家人一起入境，才可以共用一個帳號

三、註冊 Visit Japan Web 帳號

  1. 前往Visit Japan Web官方網站

    !!注意!! 目前官方只有開放網頁版，沒有APP。

  1. 點選「建立新帳號」
  1. 同意規範，下一步
  1. 輸入信箱和密碼（這是未來登入用的務必熟記）

    密碼要求至少10位數，需要有大小寫+數字+符號

異常案例：
如果也有遇到像我一樣全部輸入完畢後無法按下一步的人，請使用瀏覽器無痕模式再試一次，有看到機器人驗證才是正常的。

  1. 驗證電子郵件

到email收驗證碼的信件，回到註冊頁面並輸入驗證碼，點擊創建新帳號。

  1. 成功後點擊按鈕前往登入頁面
  1. 可選擇是否開啟2FA雙重驗證 (帳戶會比較安全)

接下來，就進入登入後的畫面：

四、建立個人基本資料

點擊登錄使用者區塊下的本人資料

  1. 第一頁全部勾 否
  1. 第二頁勾 自己輸入

  1. 填寫護照資訊和基本資料後下一步

    國籍選台灣
    家庭住址：國名 填 TAIWAN
    家庭住址：城市名 填你的居住城市

  2. 最後會再顯示一次資訊，確認無誤後點擊登錄

  3. 確認VISA資訊，短期旅遊不會有VISA，選否之後點擊返回到主畫面

若有同行家人填寫方式同上

五、填寫日本旅遊資訊

  1. 點擊 新登錄入境/返回日本預定
  1. 選擇不引用，前往登記，下一步
  1. 填寫旅遊資訊，請根據你的預定行程填寫
  • 旅行名稱：自定義好辨別的名稱
  • 抵達日本預訂日期：飛機抵達日本的日期
  • 搭乘航班號：可以看機票上寫的填寫，像我是搭長榮可以看Flight欄位，代號是BR
  • 航班號：看機票資訊，長榮為Flight欄位接在BR後面的號碼
    重點注意
  • 出發地：如果你是從桃園國際機場出發，務必填寫TAIPEI，並沒有桃園的選項。
  1. 填寫住宿資訊
    可以從Google Map查詢飯店資訊填上對應欄位即可。
    聯絡電話可留飯店的電話號碼
  1. 最後會再顯示一次資訊，確認無誤後點擊登錄預定

六、填寫入境與海關申報資訊

  1. 往下拉，點擊入境審查及海關申報

  1. 會自動代入之前填寫的內容，確認無誤就下一步

  2. 入境目的選旅遊並填寫你預計玩的天數

  1. 接著，全部選否，並確保沒有攜帶這些東西
  1. 最後再次確認並勾選以上輸入的內容與事實相符無誤後點擊登錄
  1. 接著就會有顯示QR碼的按鈕可以按
  1. 將最終的QR CODE截圖，每個人的手機都要有自己的那份QR Code

七、抵達日本後怎麼用？

入境時，如果你有QR Code會有當地人員引導到Kiosk機台。
建議先準備好QR Code畫面方便人員引導
到機台後，只要掃描護照+QRCode+指紋，接著會得到一個英文字母(a-d)。

記好字母到下一關，櫃檯上會顯示英文字母，再按照你拿到的字母排指定櫃檯即可。
一樣會需要出示護照給人員，這邊建議將口罩移除，方便人員幫忙拍照。

順利通過後，就是到等行李的關卡。
這之後就用不到QR Code了。

八、常見問題（FAQ）

Q1：哪些機場可以使用 Visit Japan Web？

A：目前日本主要的國際機場皆支援 Visit Japan Web 快速通關，包含：

  • 成田國際機場
  • 羽田機場
  • 關西國際機場
  • 中部國際機場
  • 福岡機場
  • 札幌新千歲機場
  • 沖繩那霸機場

Q2：Visit Japan Web 有 App 嗎？

A：目前沒有官方 App，僅提供網頁版。
建議使用電腦或手機瀏覽器（Chrome、Safari、Edge）登入操作。
若在網路上看到號稱「Visit Japan Web App」的程式，請小心詐騙！

Q3：同行家人或小孩需要另外登錄嗎？

A：可使用同一帳號幫家人一起申報，但必須符合以下條件：

  • 一起入境與回國
  • 每人都需填寫自己的基本資料與護照資訊
  • 系統會為每位成員生成專屬QR Code

Q4：如果資料填錯了怎麼辦？

A：發現填寫錯誤（例如航班、住宿或姓名）時，請：
1. 登入 Visit Japan Web 首頁
2. 找到該筆旅遊紀錄
3. 點「編輯」修改後重新保存
系統會自動更新 QR Code，無需重新註冊帳號。

Q5：更改航班或住宿需要重新填寫嗎？

A：若更改航班日期、班號或住宿地點，請重新編輯該入境紀錄並保存。

Q6：入境時手機沒電或沒網路怎麼辦？

A：建議提前截圖或列印QR Code備份。

Q7：換新護照後需要重新申報嗎？

A：需要。護照號碼不同代表新的身份資訊，必須修改個人資料。

Q8：每次去日本都要重新登錄嗎？

A：是的。
每次入境前都需建立新的「入境／返回日本預定」紀錄。
不過，個人與家人基本資料會自動保存，下次只需修改航班與住宿即可。

九、注意事項

  • 建議出發前先填好，不會到時候手忙腳亂
  • 如果從桃園機場起飛，出發地務必填寫TAIPEI
  • 可以截圖，或是列印出QR Code
  • 官方網站支援日文、英文、中文、韓文和越南語
