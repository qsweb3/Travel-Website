[2025] 介紹關西機場至大阪難波 - 南海電鐵特急rapi:t

簡介

本文介紹從關西機場出境後，搭乘南海電鐵前往大阪難波

南海電鐵特色

  1. 連接關西國際機場與大阪難波，是往返機場最快的方式
  2. 車分為α跟β，Rapi:t α最快只需約34分鐘即可抵達難波，β 是停比較多站，所以到難波速度比較慢一點
  3. 全車指定席，每節車廂的出口處都有一個行李區可以放行李 (記得要提早排隊，不然被搶空就沒位置)
  4. 搭乘前 5 分鐘都還可以改座位

購票地方

網路購票可以參考底下兩個連結

  • kkday - 大阪關西機場 KIX 南海電鐵機場快線 Rapi:t 電子車票
  • klook - 關西機場 - 南海電鐵特急Rapi:t 車票

電子票券兌換

當你是透過 kkday or klook 這類平台購買時，買完會給你一個連結如下圖

點選上面的連結之後，如下圖先選位置，位置選完仍可重選 (最多改 2 次)

搭車前五分鐘都可以再改位置

上圖點進去，就是下圖可以選位置和要搭乘的時間

我們當時飛機抵達關西機場是 10:15 a.m. ，因為怕趕不上就先選 11:30 的車，結果抵達南海電鐵是 10:52 ，所以就直接改選 11:00 的南海電鐵

要特別注意的地方是，這個 QR Code 需要當天才會顯示，所以選完位置之後，按下使用發現啥都沒有是正常的

等到要搭乘的那天凌晨 4 點過後，就會看到 qr code 出來，可以拿來掃描進車站

官方說明印出來沒效，一定要開手機畫面出來掃描

位置選擇建議

選位訣竅：

  • 單數車廂 ⇨ 選車尾
  • 偶數車廂 ⇨ 選車頭

14車是普通座位，56車是豪華座位(比較貴)
我們這邊以1~4車的範例座位圖如下：
行李放置區會在門的地方，所以建議位置可以離門口近一點，方便拿行李和下車，因為一個車廂只會有一個車門。

機場 -> 南海地鐵

如果下飛機是在關西機場第二航廈，則需要先前往第一航廈，而如果是在第一航廈，可以直接出境拿完行李後，會是在一樓大廳

直接去關西機場第一航廈 2 樓，照著下圖圈起來的鐵道這個指示走

到 2 樓之後，依照指示會走外面的空橋，走到底就會看到南海電鐵了，也就是下圖的 NANKAI

如果是透過 KKDAY or Klook 購買的，這時候就可以拿起手機打開這些平台給的連結，會有一個 QR Code ，走過去掃描即可

通過後要坐手扶梯往下，之後找底下這張圖片的就是南海特急列車

之後坐上列車，抵達目的難波站之後，一樣開手機顯示 qr code 掃描出站
